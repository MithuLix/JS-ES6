Escape characters

Because String must be written withion quotes, Javascript will misunderstand the string:

var x = "We are so-called"Vikings" from the north";

The string will be chopped to "We are so-called"

The solution to avoid this problem, is to use the backslash escape character.

The backslash ( \ ) escape character turns special characters into string characters:

Code Result Description
\' ' Single quote
\" " Double quote
\\ \ Backslash

Example

The escape sequence ( \', \", \\ ) inserts a single, double, backslash quote in a string.

Six other escape sequences are valid in Javascript:

Code result
\b Backspace
\f From Feed
\n NewLine
\r Carriage return
\t Horizontal Tabular
\v Vertical Tabular

The 6 characters avobe were originally designed to control typewriters, teletypes, and fax machines. They do not make any sense in HTML.

The two methods, indexOf() and search(), are equal?

The two methods are NOT equal. These are the differences:

  • The indexOf() method cannot take powerful search values (regular expressions).
